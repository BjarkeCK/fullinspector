Detailed Description

Provides a view of an arbitrary type that unifies a number of discrete concepts in the CLR. Arrays and Collection types have special support, but their APIs are unified by the TypeMetadata so that they can be treated as if they were a regular type.

This partial implementation contains methods which are used to determine if a MemberInfo can be serialized by Unity or if it can be serialized by Full Inspector.

object FullInspector.Internal.InspectedType.CreateInstance ( )

Creates a new instance of the type that this metadata points back to. If this type has a default constructor, then Activator.CreateInstance will be used to construct the type (or Array.CreateInstance if it an array). Otherwise, an uninitialized object created via FormatterServices.GetSafeUninitializedObject is used to construct the instance.
