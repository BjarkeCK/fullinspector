void InvalidateCache()
For performance reasons, the DictionaryAdapter stores an array version of the dictionary. If the adapted dictionary has been structurally modified, for example, an item has been added, then the local cache is invalid. Calling this method updates the cache, which will restore proper adapter semantics.
