Detailed Description

A property editor is the core editing abstraction used within FI. This allows for overriding of the default inspector logic for elements of a custom type. PropertyEditors are extremely similar to Unity's PropertyDrawers, except that they support generics and are fully type-safe. The AttributePropertyEditor is identical to the PropertyEditor, except that it is activated when a user adds an attribute to a type.

Recall that to get FI to actually use the property editor, you need to add the [CustomAttributePropertyEditor] attribute to the type with appropriate parameters.
