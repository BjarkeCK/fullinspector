Detailed Description

An annotation that signals that a class is a property editor for a given property type.

If the property editor is editing a generic type, then PropertyType should just be an open generic type reflecting the edited type. For example, for a List property editor, PropertyType should be typeof(List{}) (where {} are angle brackets).
