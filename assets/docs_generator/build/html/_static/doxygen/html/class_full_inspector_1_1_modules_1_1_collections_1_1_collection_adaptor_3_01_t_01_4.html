Determines whether an item can be reordered by dragging mouse.

This should be a light-weight method since it will be used to determine whether grab handle should be included for each item in a reorderable list.

Please note that returning a value of false does not prevent movement on list item since other draggable items can be moved around it.

void FullInspector.Modules.Collections.CollectionAdaptor< T >.InvalidateCache ( )

For performance reasons, the CollectionAdaptor stores an array version of the collection. If the adapted collection has been structurally modified, for example, an item has been added, then the local cache is invalid. Calling this method updates the cache, which will restore proper adapter semantics.
