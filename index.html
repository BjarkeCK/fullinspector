Full Inspector

Unity Inspector support for all .NET types

Inspect Everything

Wouldn't it be awesome if the inspector supported:

  • interfaces/abstract types (with the implementation selected in the inspector)
  • structs
  • dictionaries
  • arbitrary generic types
  • properties
  • a proper array / list editor (using Rotorz)
  • tooltips and comments via attributes on fields/properties/types

With Full Inspector, it does! Further, you get clean and painless Json.NET integration.

Guidelines

These are a couple of guidelines to keep in mind when using Full Inspector. Don't worry, they're simple and non-intrusive.

  • Derive from BaseBehavior, not MonoBehaviour
  • If you override Awake, immediately call base.Awake()
  • Annotate types with [JsonObject(MemberSerialization.OptIn)] and members with [JsonProperty]
  • Wrap Component references with Ref<>

If you have a save game system with special serialization logic, make sure you call FullInspectorSaveManager.SaveAll() before serialization.
